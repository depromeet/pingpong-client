## 설치

yarn berry를 사용하고 있습니다.

```
yarn install
yarn dev
```

## commit convention

깃모지를 사용하고 있습니다.  
노션 코드 컨벤션 확인 후, 아래 명령어로 commit 해주세요.

```
gitmoji -c
```

## storybook

storybook을 실행하고 싶으시면 아래 명령어를 입력해주세요.

```
yarn storybook
```
